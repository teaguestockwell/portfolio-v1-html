Embedding Mobile Apps

The web is an exciting platform. It allows you to reach more people than ever imaginable. I would like to show you how to deploy an application to any platform under one codebase using Flutter. Flutter is a framework by Google that uses the Skia engine to compile applications from dart into native code for Android, iPhone, Mac, Windows, and Web. It can even be embedded into an existing website.

This application is a rework of an Android App for aircraft weight and balance. It was built using the Dart language. Dart handles the logic of the application. Flutter on the other hand has UI components that can be placed inside the Dart code. The application and its UI components are compiled into HTML, CSS, and JS. The native web components are then hosted on Google Firebase. When the application runs, it sends an asynchronous request to Google Firestore. Firestore is a NoSQL database that stores documents containing Aircraft. Each aircraft contains weights and balance information, cargo configuration, and a glossary.
